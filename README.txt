UnlockCam.lua v1.0 for PES 2021
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
This module enables "Field Side" and "Live Broadcast" cameras, when playing
with 2 or more human players on the same side (Co-op/Exhibition/League/Cup/ML)


HOW TO INSTALL:
===============
1) copy UnlockCam.lua into "modules" folder of your sider
2) modify sider.ini to enable the module:

lua.module = "UnlockCam.lua"
